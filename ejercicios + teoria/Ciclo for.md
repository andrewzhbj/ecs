# Ciclo for
Un ciclo for es una estructura de control iterativa, que nos permite ejecutar de manera repetitiva un bloque de instrucciones, conociendo previamente un valor de inicio, un tama√±o de paso (si aumenta o decrece..) y un valor final para el ciclo.

## Sintaxis ##

```c
for(int i = valor inicial; i <= valor final; i++ )
{
    Bloque de Instrucciones...
}
```
